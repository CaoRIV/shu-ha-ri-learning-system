## Code Review Checklist

```markdown
# Review Checklist

- [ ] Code giải quyết đúng problem đã nêu
- [ ] Tên biến/function/module rõ nghĩa
- [ ] Không có abstraction thừa
- [ ] Edge cases quan trọng được test
- [ ] Error handling rõ ràng
- [ ] Logging/observability phù hợp
- [ ] Security/privacy risk được cân nhắc
- [ ] README/docs đủ để chạy và bảo trì
```
